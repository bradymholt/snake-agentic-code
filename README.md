# snake-agentic-code

This repo combines three HTML-based Snake game variants generated by different AI coding agents.

## Directory layout

- `codex/` - Snake game generated by Codex (`gpt-5.3-codex`)
- `claude/` - Snake game generated by Claude (`opus-4.6-claude`)
- `gemini/` - Snake game generated by Gemini (`gemini-3`)

## Prompt

> "Generate a HTML based snake game"

## Run locally

Open `index.html` directly in a browser, or serve the folder with a static server.

## GitHub Pages deployment

This repo includes `.github/workflows/pages.yml` to deploy automatically on pushes to `main`.

### One-time GitHub setup

1. Create a new GitHub repository (for example `snake-llm`).
2. Push this folder to the `main` branch.
3. In GitHub: `Settings -> Pages -> Build and deployment`, set `Source` to **GitHub Actions**.

After that, each push to `main` publishes the site.
